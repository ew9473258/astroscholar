# Astroscholar
An AI-powered research app intended for use by psychic professionals for quick retrieval of astrology-related information from historical sources, extracted using an LLM. 

- Allows users to ask questions with a simple and clean interface
- Automatically groups information by source
- Resizable window to fit alongside client conversations or other work
- Uses a PySide6 interface (in Python)
- Uses Pydantic and LangChain to process and structure LLM output


**Instructions for use:**
1. Clone or download the repository files
2. Create a .env as per the .env.example file and insert your own Groq API key
3. Install Python dependencies: 'pip install -r requirements.txt'
4. Run the app: 'python main.py'

Note: Can run without a valid API key for demonstration purposes but will not display LLM responses.
